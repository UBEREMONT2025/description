# Статусы (МВП)


Ниже — согласованный набор статусов для ключевых сущностей МВП.

Источник правды по статусам: **Bitrix24**.


## 1) Тендер (дизайнер / управляющий)

**Статусы:**

- Черновик

- Опубликован

- Сбор откликов

- Рассмотрение откликов

- Выбран исполнитель

- Договор на подписании

- Закрыт (успешно)

- Отменён


**Ключевые переходы:**

Черновик → Опубликован → Сбор откликов → Рассмотрение → Выбран исполнитель

→ Договор на подписании → Закрыт


## 2) Этап дизайн‑проекта

**Статусы:**

- Запланирован

- В работе

- Передан на проверку

- На замечаниях

- Доработка

- Принят

- Закрыт


**Ключевые переходы:**

Запланирован → В работе → Передан на проверку

→ (На замечаниях → Доработка → Передан на проверку)*

→ Принят → Закрыт


## 3) Акт (этап/финал)

**Статусы:**

- Черновик

- Отправлен на подпись

- Подписан

- Отклонён

- Отменён


**Ключевые переходы:**

Черновик → Отправлен на подпись → Подписан

или Черновик → Отправлен на подпись → Отклонён → Черновик


## 4) Платёж по акту

**Статусы:**

- Не выставлено

- Ожидается предоплата (50%)

- Оплачено 50%

- Ожидается постоплата (50%)

- Оплачено 100%

- Просрочено

- Отменено


**Ключевые переходы:**

Не выставлено → Ожидается предоплата → Оплачено 50%

→ Ожидается постоплата → Оплачено 100%


## 5) Замечания (отдельный процесс)

**Статусы:**

- Новое

- В работе

- Устранено

- Подтверждено

- Закрыто
